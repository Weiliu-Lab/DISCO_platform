# Planner-Expert Case for Bistable Adsorption Discovery

Bistable adsorption is an important but often overlooked motif in heterogeneous catalysis because multiple locally stable adsorption states can reshape the adsorption free-energy landscape, alter kinetic accessibility, and even redirect the dominant reaction channel under operating conditions. In particular, when a large aromatic adsorbate such as benzene interacts with a close-packed metal surface, the competition between dispersion-driven flat adsorption and more weakly bound tilted or lifted configurations can generate multiple metastable basins on the potential energy surface. This makes bistable adsorption a suitable benchmark for testing whether an autonomous agent can move beyond single-site enumeration and instead reason across modeling, structure search, and post-analysis stages. Following the discussion of adsorption-state complexity and competing minima highlighted in the JACS study cited above, we therefore redesigned our demonstration around the question: can a Planner-Expert agent autonomously identify evidence for bistable adsorption of benzene on Pt(111), while also preparing the system for subsequent pulling-curve analysis?

To answer this question, we frame the user request in a more agent-native way: explore whether benzene on Pt(111) exhibits two distinct low-energy adsorption states, and if so, build the adsorption models, optimize them, and prepare the workflow for a stretching-curve analysis. The Planner first interprets the scientific intent and dispatches expert subtasks. The molecule expert extracts the adsorbate name from the prompt, converts it into a canonical SMILES representation, and generates a three-dimensional molecular model using RDKit. In parallel, the surface expert identifies the substrate, crystal prototype, and Miller index from the same prompt, retrieves the bulk Pt structure from the Materials Project database, analyzes its symmetry information, and cleaves the Pt(111) slab. The assembly expert then combines the benzene adsorbate and Pt(111) slab into multiple chemically meaningful initial guesses by varying adsorption sites and molecular orientations. On top of these initial candidates, the search expert applies a potential-accelerated minima-hopping style exploration to uncover both low-energy and metastable adsorption states, thereby creating a more realistic test of agentic reasoning than a single fixed adsorption geometry. Finally, we reserve a dedicated stretch expert that will call our custom pulling function and post-processing plotter in the next demo, allowing the present case to serve as the structural and workflow foundation for the full stretching-curve analysis.

From the perspective of agent design, this example better matches the intended Planner-Expert architecture than our previous simple demonstration. The Planner is responsible for global decomposition and tool routing, whereas each Expert operates on a well-defined scientific subproblem: molecular structure generation, surface construction, adsorption assembly, metastable-state search, and later stretching analysis. The value of the case is not only that it targets a scientifically meaningful bistability question, but also that it requires cross-domain coordination between natural-language parsing, materials database retrieval, atomistic model construction, approximate structure search, and downstream analysis. As a result, the benchmark more faithfully reflects the kind of multi-step research assistance that an autonomous computational chemistry agent is expected to provide.
